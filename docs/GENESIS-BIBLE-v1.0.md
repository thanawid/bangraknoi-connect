# 📖 Bangraknoi Connect --- Genesis Bible v1.0

> รัฐธรรมนูญของโปรเจกต์ (Project Constitution)

------------------------------------------------------------------------

# 1. Project DNA

เรา **ไม่ได้สร้างเว็บไซต์**

เรา **กำลังสร้างเมืองการ์ตูนที่บังเอิญเป็นเว็บ**

เป้าหมายคือ

> เมื่อคนเปิดหน้าแรกแล้วพูดว่า

> **"ใช่...นี่บ้านเรา"**

------------------------------------------------------------------------

# 2. Core Philosophy

1.  เมืองคืออินเทอร์เฟซ (The City is the Interface)
2.  เมืองเล่าเรื่อง ข้อมูลจะโผล่เมื่อผู้ใช้ต้องการ
3.  ทุกอาคารคือประตูสู่ข้อมูล
4.  ระบบต้องกลืนไปกับเมือง
5.  ผู้ใช้กำลัง "เดินเข้ามาในบางรักน้อย" ไม่ใช่เปิดเว็บ

------------------------------------------------------------------------

# 3. สิ่งที่ห้ามหลุด

-   ห้าม Dashboard
-   ห้าม Card ใหญ่บังเมือง
-   ห้าม Popup กลางจอ
-   ห้ามอธิบายทุกอย่างด้วยข้อความ
-   ห้ามทำให้เมืองกลายเป็นพื้นหลัง

ถ้าสิ่งใดทำให้เมืองถูกบัง → ออกแบบใหม่

# 5. World Bible

แกนเมือง = คลองอ้อมนนท์

Landmark หลัก

-   วัดบางรักน้อย
-   พระใหญ่
-   เทศบาลเมืองบางรักน้อย
-   รถไฟฟ้าสายสีม่วง
-   ตลาด
-   ร้านอาหาร
-   ร้านช่าง
-   สวนทุเรียน
-   สวนมะม่วง
-   บ้านเรือนชุมชน

ตำแหน่งไม่จำเป็นต้องตรงจริง 100%

แต่ "ความรู้สึก" ต้องใช่

------------------------------------------------------------------------

# 6. Living City

เมืองต้องมีชีวิตเสมอ

-   เมฆ
-   นก
-   น้ำ
-   เรือ
-   รถไฟฟ้า
-   คนเดิน
-   ใบไม้ไหว
-   ควันร้านอาหาร

BC ไม่ยืนเฉย

BC เดินเล่น ให้อาหารปลา เข้าร้านกาแฟ โบกมือ ใช้ชีวิตเหมือนคนในเมือง

------------------------------------------------------------------------

# 7. Time System

ไม่มีปุ่ม Day/Night

ใช้เวลาจริงของเครื่องผู้ใช้

เช้า สาย บ่าย เย็น กลางคืน

เมืองเปลี่ยนเอง

------------------------------------------------------------------------

# 8. Seasonal System

หน้าทุเรียน หน้ามะม่วง สงกรานต์ ลอยกระทง ปีใหม่

เมืองเปลี่ยนตามฤดูกาล

------------------------------------------------------------------------

# 9. UX Rules

Menu = Place

ร้านอาหาร = ร้านจริง

ช่าง = บ้านช่าง

ตลาด = ตลาดจริง

บริการ = อาคารเทศบาล

ผู้ใช้เดินเข้าสถานที่ ไม่ใช่กดปุ่ม

------------------------------------------------------------------------

# 10. Roadmap

Genesis V1 - Master Scene - Living Animation

Genesis V2 - เมืองเป็นเมนู

Genesis V3 - ข้อมูลจริง - ร้านจริง - ช่างจริง - ข่าวจริง

Genesis V4 - เมืองเติบโตจากข้อมูลของชุมชน

------------------------------------------------------------------------
