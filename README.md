# Bangraknoi Connect — Play Map V1

โฟกัสรอบนี้: สร้างเว็บจาก Master Scene โดยให้เมืองเป็นอินเทอร์เฟซ

## มีอะไรในเวอร์ชันนี้
- ใช้ Play Map Master Scene เป็นหน้าแรกเต็มจอ
- โลโก้เล็ก มุมซ้าย ไม่บังเมือง
- กลางวัน/กลางคืนตามเวลาจริง
- เอฟเฟกต์เมืองมีชีวิต: เมฆ นก น้ำ รถไฟฟ้า ควันร้าน ใบไม้
- 8 หมวดเป็นพื้นที่ในเมือง / hotspot
- แตะพื้นที่แล้วข้อมูลค่อยโผล่
- ไม่มีกรอบเหลือง
- ยังไม่ต่อฐานข้อมูลจริง
