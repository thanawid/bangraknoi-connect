# บางรักน้อย Connect — Community Portal V5

เว็บชุมชนรูปแบบใหม่ตามภาพอ้างอิง โทนสีม่วง–ส้ม–ทอง เน้นค้นหาข้อมูลและเข้าถึงบริการได้รวดเร็วทั้งเดสก์ท็อปและมือถือ

## ฟีเจอร์

- ฮีโร่เมืองบางรักน้อยพร้อม BC Mascot
- ค้นหาข้อมูลทั้งเว็บและกรองตามหมวดหมู่
- หมวดบริการ 8 หมวด พร้อมแผงรายละเอียดและลิงก์ใช้งานจริง
- การ์ดแนะนำสถานที่ ร้านอาหาร ช่าง ข่าว และหลักสูตรฝึกอาชีพ
- ลิงก์แจ้งเรื่อง โทรศัพท์ แผนที่ ข่าว ปฏิทิน LINE และ Messenger
- เมนูมือถือ หน้าต่างเข้าร่วมชุมชน และรองรับคีย์บอร์ด
- ติดตั้งเป็น PWA และเปิดข้อมูลเดิมแบบออฟไลน์ได้

## วิธีเปิดบน Windows

ดับเบิลคลิก `START-BANGRAKNOI-CONNECT.bat` ระบบจะเปิดเว็บในเบราว์เซอร์ให้โดยอัตโนมัติ ปิดหน้าต่างคำสั่งเมื่อต้องการหยุดเว็บ

หรือใช้เว็บเซิร์ฟเวอร์อื่น:

```powershell
python -m http.server 4173
```

จากนั้นเปิด `http://localhost:4173`

## แก้ไขข้อมูล

- ข้อมูลหมวดหมู่และลิงก์บริการ: `data.js`
- โครงหน้า: `index.html`
- รูปแบบและสี: `styles.css`
- การค้นหาและปฏิสัมพันธ์: `app.js`
