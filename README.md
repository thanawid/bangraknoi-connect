# บางรักน้อย Connect — Scene First Community V1

เวอร์ชันนี้กลับมาใช้ฐาน V1/V8 ของผู้ใช้เป็นแกนภาพและมาสคอต แต่ปรับแนวทางเป็น “ฉากเว็บชุมชน” ตามคอนเซปต์ Nintendo + Disney + LINE + Google Maps

## มีอะไรในชุดนี้
- หน้าแรกแบบฉาก 2D ใช้ภาพ `hero_bg.png`
- Hotspot ในฉาก: ร้านอาหาร / หาช่าง / ฝึกอาชีพ / ข่าวสาร / บริการชุมชน / สินค้าท้องถิ่น
- Animation เมฆ น้ำ มาสคอต และผู้ช่วยชุมชน
- ระบบค้นหาและกรองหมวดหมู่
- ฟอร์มสมัครเข้าระบบ
- หลังบ้าน `admin.html` สำหรับอนุมัติ / ไม่อนุมัติ / ลบข้อมูล
- ข้อมูลเดโมเก็บใน localStorage ใช้งานได้ทันทีโดยไม่ต้องมี Server
- มี `Code.gs` เดิมสำหรับต่อ Google Sheet ภายหลัง

## วิธีใช้
1. เปิด `index.html` เพื่อดูหน้าเว็บ
2. กด “สมัครเข้าร่วม” เพื่อส่งข้อมูลร้าน/ช่าง/กลุ่มอาชีพ
3. เปิด `admin.html` เพื่ออนุมัติข้อมูล
4. กลับมาหน้าเว็บ รายการที่อนุมัติจะแสดงในหน้าเว็บ

## แนวทางต่อไป
- ต่อ Google Sheet/Apps Script ให้ข้อมูลเป็นฐานกลาง
- เพิ่ม LINE OA สำหรับเปิดจากมือถือ
- เพิ่มแผนที่และค้นหาใกล้ฉัน
- เพิ่ม AI ผู้ช่วยบางรักน้อย
- ทำ PWA ให้ติดตั้งบนมือถือได้
